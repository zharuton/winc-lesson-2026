# Phase 4: 自分だけの道を極める
## 〜 専門ルートの選択と完成へ 〜

---

## 1. これまでの振り返り

- **Phase 1:** HTML/CSS で「外装」を作った
- **Phase 2:** Vanilla JS で「命」を吹き込んだ
- **Phase 3:** Next.js で「エンジン」を載せ替えた

**今日からは、この最強の基盤の上に「究極のパーツ」を載せていきます。**

---

## 2. 選べる3つの専門ルート

### A: Tech News Hub
- 情報収集を自動化する。
- API 統合、フィルタリング、データ永続化。

### B: Three.js Design
- 圧倒的な視覚表現を追求する。
- 3D空間、ライティング、アニメーション。

### C: Simple Web Game
- 遊び心とロジックを両立させる。
- ゲームループ、当たり判定、高度な状態管理。

---

## 3. 第14週のミッション

各ルートの「スターターコード」を `project-core` に統合し、動かす。

- **News:** データの型を定義し、最初のフェッチを成功させる。
- **Three.js:** Canvas を画面に表示し、3Dの立方体を置く。
- **Game:** プレイヤーの操作を受け付け、画面内で動かす。

---

## 4. 最終目標: デモデイ

**第17週の最後に、完成したダッシュボードを発表します。**

- 単なる「課題」ではなく、**「自分が毎日使いたいツール」**になっているか？
- コードが美しく、他人が読めるようになっているか？
- Vercel にデプロイされ、世界中に公開されているか？

---

## 5. さあ、始めよう！

どのルートに進むか決まりましたか？

**自分の情熱が一番湧くものを選んでください。**
エンジニアにとって、一番の成長は「作りたいものを作る」瞬間に訪れます。
